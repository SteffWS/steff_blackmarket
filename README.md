# Steff Blackmarket

A fully customizable, secure black market script built for **QBCore** using **ox_lib**, **ox_target**, and **ox_inventory**.

## üìò Overview

Steff Blackmarket enables immersive criminal trading with:

- A **PED vendor** that randomly relocates
- **Drops** of ordered items with **lock codes**
- Cart-based checkout system
- Payments via **cash** or **black_money** (bank **disabled by default**; optional with phone)
- **Localization support** via locale files
- **Discord webhook** logging of purchases and collections
- Full server-side price validation and cooldowns for security
